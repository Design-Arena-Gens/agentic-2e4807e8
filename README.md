## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the bot.

## Tools

The bot can autonomously detect when to use tools based on user input:

- **Calculate**: Perform mathematical operations
- **Get Time**: Retrieve current date and time
- **Random Number**: Generate random numbers in a range
- **Word Count**: Count words and characters in text
